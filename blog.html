Q . What is Document Object Model(DOM)?


Ans : The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

Q . how do you select an element from the DOM ?


Ans :To select an element from the DOM (Document Object Model) in JavaScript, number of methods can be used like getElementById,getElementsByClassName,querySelector.

Q . what is event delegation in the context of the dom and why is it useful ?


Ans :Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.

Q . How do you manipulate an element's attributes and styles using the dom ?


Ans :To manipulate an element's attributes and styles using the DOM (Document Object Model) in JavaScript, you first need to select the desired element using methods like document.getElementById or document.querySelector, then use the setAttribute method to change attributes and the style property to modify CSS styles directly on the element.
